chewing-grid.css

The chewing-grid is the ideal CSS Grid for card listing design.

Flexbox FTW

It is made with flexbox and a lot of love.
No need for custom media-queries to configure it!
So it could be used easily with any layout.

Atomic classes or Sass

You can use it with atomic classes, custom CSS rules or Sass mixins.
It's all documented in the project README.

Live demo

Use the right bars to resize the demonstration.
